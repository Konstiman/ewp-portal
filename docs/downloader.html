DESCRIPTION

Modul slouzi ke stazeni a ulozeni XML katalogu instituci Erasmus Without Paper.

ATTRIBUTES

status : Str

Reprezentuje stav downloaderu. Pripustne jsou hodnoty 'ok', 'error' a 'warning'.

status : Str

Textovy komentar stavu downloaderu.
