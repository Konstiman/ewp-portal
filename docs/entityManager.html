NAME

Manager::EntityManager

DESCRIPTION

Modul pro ukladani EWP entit do databaze.

ATTRIBUTES

dbh : DBI

Povinny atribut pro ulozeni instance databaze.

METHODS

clearDatabase () : None

Smaze obsah neciselnikovych tabulek v db.

getCountryId ( code : Str ) : Int

Vraci id zeme podle kodu, pokud je zeme s timto kodem ulozena v databazi.

Hleda v tabulce `country`.

saveCountryCode ( code : Str ) : Int

Ulozi kod zeme do databaze do tabulky `country`. Vraci id nove vlozeneho zaznamu.

saveAddress ( address : Entity::Address ) : Entity::Address

Ulozi objekt adresy do databaze do tabulky `address`. Objektu nastavi id nove vlozeneho zaznamu.

saveInstitution ( institution : Entity::Institution ) : Entity::Institution

Ulozi objekt instituce do databaze do tabulky `institution` a navazujicich tabulek. Objektu nastavi id nove vlozeneho zaznamu.

saveContact ( contact : Entity::Contact ) : Entity::Contact

Ulozi objekt kontaktu do databaze do tabulky `contact` a navazujicich podtabulek. Objektu nastavi id nove vlozeneho zaznamu.

saveUnit ( unit : Entity::Unit ) : Entity::Unit

Ulozi objekt organizacni jednotky do databaze do tabulky `organizational_unit` a navazujicich tabulek. Objektu nastavi id nove vlozeneho zaznamu.

saveOpportunity ( opportunity : Entity::LearningOpportunity ) : Entity::LearningOpportunity

Ulozi objekt studijni prilezitosti do databaze do tabulky `learning_opportunity` a navazujicich tabulek. Objektu nastavi id nove vlozeneho zaznamu.

saveIndex (institution : Entity::Institution, units : ArrayRef[Entity::Unit], courses : ArrayRef[Entity::LearningOpportunity]) : Bool

Z objektu instituce a k ni souvisejicich jednotek a kurzu vytvori a ulozi textovou reprezentaci instituce, ktera je nasledne pouzita pro fulltextove vyhledavani na Studentskem portalu.

createIndex () : Bool

Vytvori fulltext index pro vyhledavani v institucich.
