NAME

StudentPortal::Model::DBIModel - DBI Model Class

DESCRIPTION

DBI Model Class. Hlavni datovy model pro Student Portal.

METHODS

simpleSelect (query : Str, parameters : Array[Str]) : ArrayRef[HashRef[Str]]

Obalova metoda pro select z databaze, ktery se vola ve vsech metodach stejne.

getInstitutionsListData () : ArrayRef[HashRef[Str]]

Vraci seznam vsech instituci a zakladni informace o nich: id, identifier, names, abbreviation, logo, locality, country.

Jedinym nepovinnym parametrem je HashRef s parametry filtrovani.

getInstitutionCountriesData () : ArrayRef[HashRef[Str]]

Vraci pole vsech statu, ve kterych je nejaka instituce zobrazitelna portalem. Zaznam obsahuje nazev zeme a jeji id.

getAddress ( id : Int ) : HashRef[Str]

Vraci HashRef s informacemi o jedne konkretni adrese.

getInstitutionInformation ( identifier : Str ) : HashRef[Item]

Vraci HashRef s informacemi o jedne konkretni instituci.

getInstitutionCities () : ArrayRef[HashRef[Str]]

Vraci pole s informacemi o adresach vsech instituci. Soucasti kazdeho zaznamu je mesto a stat, kde instituce lezi, a jeji identifikator.

getContact ( id : Int ) : HashRef[Str]

Vraci HashRef s informacemi o jednom konkretnim kontaktu.
